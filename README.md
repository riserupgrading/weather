
# 🌤️ Weather App

A clean, responsive weather application built with React.js that fetches real-time weather data using the OpenWeatherMap API.

## 🚀 Live Demo
https://climateloc.netlify.app/

## 📸 Screenshot

## ✨ Features
- 🔍 Search weather by city name
- 🌡️ Real-time temperature, humidity, and wind speed
- 🎨 Dynamic weather icons based on conditions
- 💎 Glassmorphism UI design
- 📱 Fully responsive


## 🛠️ Tech Stack
- React.js (Vite)
- OpenWeatherMap API
- CSS3 (Glassmorphism)

## ⚙️ Installation & Setup

1. Clone the repository
\`\`\`bash
git clone https://github.com/riserupgrading/weather.git
cd weather
\`\`\`

2. Install dependencies
\`\`\`bash
npm install
\`\`\`

3. Create a `.env` file in root and add your API key
\`\`\`env
VITE_API_KEY=your_openweathermap_api_key
\`\`\`

4. Run the project
\`\`\`bash
npm run dev
\`\`\`
